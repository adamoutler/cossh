# QA Validation for SSH-95 and SSH-96

## Implementation Details
1. **Env Vars (SSH-95)**: Extended `ConnectionProfile` and `AddEditProfileScreen` to accept `envVars`. `SshConnectionManager` calls `session.setEnvVar` before `startShell()` or `exec()`.
2. **Port Forwarding (SSH-96)**: Extended `ConnectionProfile` to parse comma-separated `L:localPort:remoteHost:remotePort` and `R:remotePort:localHost:localPort` forwards. Updates `SshConnectionManager` to establish these tunnels via `sshClient.newLocalPortForwarder` and `remotePortForwarder.bind()` during session start. Threads and sockets are actively managed and cleaned up in `disconnect()`.
