# Verification Proof for SSH-101

The `IdentityListScreen` has been updated to include a `DisposableEffect` that triggers `viewModel.loadIdentities()` on `Lifecycle.Event.ON_RESUME`. This ensures that when returning from the `AddEditIdentityScreen` after creating or modifying an identity, the list correctly refreshes and displays the new/updated entry.

```kotlin
    val lifecycleOwner = androidx.compose.ui.platform.LocalLifecycleOwner.current
    DisposableEffect(lifecycleOwner) {
        val observer = androidx.lifecycle.LifecycleEventObserver { _, event ->
            if (event == androidx.lifecycle.Lifecycle.Event.ON_RESUME) {
                viewModel.loadIdentities()
            }
        }
        lifecycleOwner.lifecycle.addObserver(observer)
        onDispose {
            lifecycleOwner.lifecycle.removeObserver(observer)
        }
    }
```
